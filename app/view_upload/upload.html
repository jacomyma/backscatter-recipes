Backscatter Network Scripts

This tool helps you visualize network graphs in different ways.

How to use

Load your network into the tool in the lower-left corner or try the reference network. Then pick and edit a script and enjoy the outcome: a processed network, image, or anything Javascript can produce.

Note: some scripts require editing their settings so that they work with your own network. See documentation.

How to edit scripts

You can edit them in the interface or simply copy-paste them from a text editor. The language is Javascript. The graph library used is Graphology.
